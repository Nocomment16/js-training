Routing (1/7)

Configure application routes

    Create the app/app-routing.module.ts file. Export the appRoutes constant which is going to keep the application routes so that:
  • all paths starts with book-app,
  • the books path (relative to book-app) leads to the BookOverviewComponent,
  • the book path leads to the BookDetailsComponent (when adding a new book),
  • the book/:id path leads to the BookDetailsComponent (when editing an existing book),
  • the empty path ('') redirects to /book-app/books.
