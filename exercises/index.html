Exercises

Closure (1/3)

  1. create main.js file in the main directory
  2. create personFactory function in the main.js file
  3. create details object inside the personFactory function with properties: firstName, lastName, accounts list
  4. firstName, last name should have some string values assigned
  5. accounts list should have single account object inside with some example balance (number) and currency (string)

Closure (2/3)

  1. personFactory should return object with properties firstName, lastName and method sayHello
  2. firstName and lastName should take its values from inner details object
  3. sayHello method should return a string with firstName, lastName and number of accounts assigned to an object created by personFactory

Closure (3/3)

  1. create an object using personFactory function and assign it to a variable
  2. call sayHello method inside of a console.log method in order to print the result of the method
  3. attach the main.js script to the index.html file
  4. open index.html in the browser and check the results in the browser console
  5. run the script in the node.js with the following commad: node main.js - check the results
