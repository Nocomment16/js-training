JavaScript in the browser

Document Object Model - (DOM) is a cross-platform and language-independent application programming interface that treats an HTML, XHTML, or XML document as a tree structure wherein each node is an object representing a part of the document. The objects can be manipulated programmatically and any visible changes occurring as a result may then be reflected in the display of the document

Attributes vs Properties

Attributes are defined by HTML. Properties are defined by DOM.

Some HTML attributes have 1:1 mapping onto properties. id is one example of such.

Some do not (e.g. the value attribute specifies the initial value of an input, but the value property specifies the current value).

Html Attributes

Attributes are defined by HTML, all definitions inside HTML tag are attributes


<div id="someId" class="highlight" custom-attr="custom-attr"></div>
		  

document.getElementById('someId').attributes
// returns NamedNodeMap {0: id, 1: class, 2: custom-attr, length: 3}

Dom Properties

Properties belong to DOM, the nature of DOM is an object in JavaScript


<div id="someId" class="highlight" custom-attr="custom-attr"></div>
		  

// custom attribute - not reflected in HTML
document.getElementById('someId').newProp = 1; // setter
document.getElementById('someId').newProp; // getter, returns 1

// non-custom attribute - change reflected in HTML
document.getElementById('someId').className = 'bold';

Selecting nodes


document.getElementById('someId');
document.getElementsByTagName('tagName');
document.getElementsByClassName('className');

document.querySelector('cssSelector');
document.querySelectorAll('cssSelector');

// example
document.querySelector('div.user-panel.main input[name=login]');
// finds first input with name attribute which has 'login' value
// input has to be under div with classes user-panel and main

Creating / adding / deleting nodes


document.createElement(element);
document.removeChild(element);
document.appendChild(element);
document.replaceChild(element);

Editing elements


element.innerHTML =  'new content';
element.attribute = 'attr value';
element.style.property = 'new style';

Events

The Event interface represents any event which takes place in the DOM; some are user-generated (such as mouse or keyboard events), while others are generated by APIs (such as events that indicate an animation has finished running, a video has been paused, and so forth)


// example events
onclick	            // The user clicks an HTML element
onmouseover	    // The user moves the mouse over an HTML element
onmouseout	    // The user moves the mouse away from an HTML element
onkeydown	    // The user pushes a keyboard key

Reacting on events


element.onclcik = function(){
    console.log('clicked!');
};

element.addEventListener('click', function(){
    console.log('clicked');
});
